✅ Completed Tasks

Task 1: Movie Genre Classification:
A complete end-to-end NLP and Machine Learning pipeline for predicting movie genres from plot summaries.
Key Highlights:
*TF-IDF based feature engineering with n-grams
*Multiple model training and comparison:
    -Naive Bayes
    -Logistic Regression
    -Calibrated Linear SVM
*Top-K genre prediction with confidence scores
*Explainable AI (XAI) using linear model coefficients
*Detailed error analysis and confusion matrix visualization: Movie_Genre_Classification/README.md

🧠 Technical Stack

*Programming Language: Python
*Core Libraries: NumPy, Pandas
*Machine Learning: Scikit-learn
*NLP: TF-IDF Vectorization
*Visualization: Matplotlib, Seaborn
*Model Persistence: Joblib / Pickle
